# 🚀 DEVS PROJECT – AI Engineer Edition 2026

> **Una ruta completa, gratuita y estructurada para convertirte en un Ingeniero de Software con especialización en Inteligencia Artificial y Agentes.**

---

## 📖 ¿Qué es este proyecto?

Este repositorio nace con una idea muy sencilla:

> **Aprender programación no debería significar perderse entre cientos de cursos, videos y tutoriales.**

Durante años han existido excelentes plataformas como **The Odin Project** o **freeCodeCamp**, pero gran parte de su contenido está en inglés o requiere combinar demasiadas fuentes para completar una formación sólida.

Este proyecto busca ofrecer una experiencia similar, completamente organizada y documentada en español.

No es un curso.

No es una recopilación de enlaces.

Es un **plan de entrenamiento**.

Cada módulo contiene:

- 📚 Teoría cuidadosamente seleccionada.
- 💻 Ejercicios progresivos.
- 🛠️ Proyectos reales.
- ✅ Checklists.
- 📖 Recursos oficiales.
- 📝 Documentación.
- 🎯 Objetivos claros.

Al finalizar la ruta habrás construido un portafolio profesional y tendrás las bases necesarias para especializarte en Inteligencia Artificial y desarrollo de Agentes.

---

# 🎯 Objetivo Final

Convertirse en un desarrollador capaz de:

- Diseñar aplicaciones web modernas.
- Desarrollar APIs profesionales.
- Trabajar con bases de datos.
- Utilizar Docker.
- Implementar aplicaciones en la nube.
- Escribir código limpio y mantenible.
- Construir sistemas con Inteligencia Artificial.
- Desarrollar Agentes utilizando modelos de lenguaje (LLMs).
