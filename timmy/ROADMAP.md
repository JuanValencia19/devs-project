# Roadmap de aprendizaje

Este roadmap organiza la ruta de estudio en fases progresivas para llegar desde los fundamentos hasta la construcción de sistemas con IA y agentes.

## Fase 0 - Preparación

- Módulo 00: bienvenida y mentalidad.
- Módulo 01: herramientas de desarrollo.

Documentación oficial recomendada:
- Visual Studio Code Docs: https://code.visualstudio.com/docs
- Git Docs: https://git-scm.com/doc
- GitHub Docs: https://docs.github.com/

## Fase 1 - Fundamentos web

- Módulo 02: HTML.
- Módulo 03: CSS.
- Módulo 04: Git y GitHub.
- Módulo 05: JavaScript.

Documentación oficial recomendada:
- MDN Web Docs (HTML, CSS y JavaScript): https://developer.mozilla.org/
- W3C HTML Standard: https://html.spec.whatwg.org/
- JavaScript MDN: https://developer.mozilla.org/es/docs/Web/JavaScript

## Fase 2 - Backend y datos

- Módulo 06: Python.
- Módulo 07: SQL y PostgreSQL.
- Módulo 08: backend con FastAPI.

Documentación oficial recomendada:
- Python Docs: https://docs.python.org/3/
- PostgreSQL Docs: https://www.postgresql.org/docs/
- FastAPI Docs: https://fastapi.tiangolo.com/

## Fase 3 - Frontend moderno

- Módulo 09: React.

Documentación oficial recomendada:
- React Docs: https://react.dev/
- React Router Docs: https://reactrouter.com/

## Fase 4 - Calidad y despliegue

- Módulo 10: Docker.
- Módulo 11: testing.
- Módulo 12: cloud y deploy.

Documentación oficial recomendada:
- Docker Docs: https://docs.docker.com/
- Pytest Docs: https://docs.pytest.org/en/latest/
- Jest Docs: https://jestjs.io/docs/getting-started
- Vite Docs: https://vite.dev/guide/

## Fase 5 - Inteligencia Artificial

- Módulo 13: inteligencia artificial.
- Módulo 14: desarrollo de agentes.

Documentación oficial recomendada:
- OpenAI API Docs: https://platform.openai.com/docs
- Hugging Face Docs: https://huggingface.co/docs
- LangChain Docs: https://python.langchain.com/docs/

## Proyecto final

Al finalizar la ruta deberías poder:

- Construir aplicaciones web completas.
- Desarrollar APIs profesionales.
- Trabajar con bases de datos.
- Publicar aplicaciones.
- Implementar soluciones con IA y agentes.

Documentación oficial recomendada para el proyecto final:
- GitHub Docs: https://docs.github.com/
- Docker Docs: https://docs.docker.com/
- Deploy con Vercel/Render: https://vercel.com/docs y https://render.com/docs

## Regla de progreso

No pases al siguiente módulo si el módulo actual no está completado con teoría, ejercicios y proyecto.
