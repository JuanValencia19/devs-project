# DEVS PROJECT - AI Engineer Edition 2026

Esta carpeta reúne la ruta de aprendizaje propuesta para convertirte en un desarrollador de software con enfoque en Inteligencia Artificial y Agentes.

## ¿Qué encontrarás aquí?

- Un roadmap progresivo desde los fundamentos hasta IA y agentes.
- Módulos organizados por tema y con estructura consistente.
- Espacios para teoría, ejercicios, proyectos, retos y recursos.
- Un sistema de seguimiento para registrar tu progreso.

## Estructura del aprendizaje

Cada módulo incluye:

- README con objetivos del tema.
- Carpeta de teoría.
- Carpeta de ejercicios.
- Carpeta de proyectos.
- Carpeta de desafíos.
- Carpeta de recursos.
- Carpeta de notas.
- Archivo checklist.md para seguimiento.

## Metodología recomendada

1. Lee la teoría del módulo.
2. Toma apuntes en la carpeta de notas.
3. Resuelve los ejercicios.
4. Construye el proyecto propuesto.
5. Documenta lo aprendido.
6. Marca los puntos completados en el checklist.

## Ruta principal

- Módulo 00: bienvenida y mentalidad.
- Módulos 01 a 05: herramientas, HTML, CSS, Git/GitHub y JavaScript.
- Módulos 06 a 08: Python, SQL y backend con FastAPI.
- Módulo 09: React.
- Módulos 10 a 12: Docker, testing y cloud.
- Módulos 13 y 14: inteligencia artificial y desarrollo de agentes.

## Recomendación final

No avances de módulo sin completar al menos un proyecto práctico y documentarlo correctamente.

¡El objetivo no es solo aprender, sino construir una base sólida para trabajar en proyectos reales.
